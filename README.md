# Portfolio Website

This is the personal portfolio website for **Sayan Mondal**. It is a static, single-page site built to showcase education, skills, projects, certificates, and contact details in a polished responsive layout.

## Features

- Hero section with animated typing text and resume download
- Theme toggle with light and dark modes
- Smooth scrolling navigation and mobile menu
- Scroll progress indicator and back-to-top button
- About, Skills, Education, Projects, and Certificates sections
- Contact form modal powered by FormSubmit
- Responsive design with custom visuals, loader, and animated background

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Google Fonts: Inter
- FormSubmit for contact form delivery

## Project Structure

- `index.html` - Main portfolio page
- `styles.css` - All styling and responsive layout rules
- `script.js` - Theme switching, animations, navigation, and contact form behavior
- `assets/` - Images, certificates, and resume files

## How To Run

You can open `index.html` directly in a browser, or serve the folder with a local development server for the best experience.

### Option 1: Open Directly

Double-click `index.html` or open it in your browser.

### Option 2: Use a Local Server

If you have VS Code Live Server, use that extension to serve the folder.

Or run a simple Python server from the project directory:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000` in your browser.
